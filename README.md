# Documentation

Projekt testów automatycznych UI + API zbudowany z użyciem:

- Selenium WebDriver
- pytest
- requests
- Page Object Model

Testujemy demo aplikację: [https://demoqa.com/login](https://demoqa.com/login)

## ✅ Zakres testów
- Logowanie przez UI
- Weryfikacja odpowiedzi API

## 🔧 Stack
- Python 3.x
- Selenium
- pytest
- requests
